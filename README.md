## Install dependencies
```bash
npm install
```

## Compile JSX
1. windows - `./compile.bat`
2. *unix - `./compile.sh`

## Make documents
```bash
webpack
node ./docs
```
## webpack 和 babel 需要全局安装
```bash 
npm install webpack -g
npm install babel -g
```

## 需要在根目录创建 `lib` 目录，将 `react.js` 和 `react-dom.js` 放入其中，demo才能正常运行。
